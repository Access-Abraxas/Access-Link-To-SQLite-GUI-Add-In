# SQLite ODBC Driver for Windows



This repository provides pre-built Windows installer for the SQLite ODBC drivers for Windows. The drivers are originally created and maintained by [Christian Werner](http://www.ch-werner.de/) and can be downloaded directly from this repository or his website.



## About the Project



The SQLite ODBC driver allows Windows applications, like Microsoft Access, to connect to SQLite databases using the ODBC interface. These drivers are essential for integrating SQLite with ODBC-compliant tools like Microsoft Access, Microsoft .NET, or other applications that rely on ODBC for database connectivity in Windows.



You can find the original project and source code at:
[http://www.ch-werner.de/sqliteodbc/](http://www.ch-werner.de/sqliteodbc/)
